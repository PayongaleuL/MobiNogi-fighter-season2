# 🥊 마비노기 모바일 시즌2 격투가 룬 계산기
시즌 2 개편에 따른 격투가 전용 신규 룬 데이터베이스(88종)와 기존 엑셀 계산기 v2.3의 연산 공식을 이식한 반응형 웹 계산기 프로그램입니다.
## 🌟 주요 기능
1. **시즌 2 신규 룬 연동**: 스크린샷 88장으로부터 자동 추출한 정밀 룬 데이터 탑재 (무기, 방어구, 장신구, 엠블럼)
2. **실시간 DPS 시뮬레이션**: 마을 공격력, 치명타 수치 및 스킬 개조 단계 변경 시 즉각적인 DPS 예측 연산
3. **조건부 가동률 슬라이더**: '거대한 분노', '무너진 경계', '초월', '백금 천칭' 등 조건부로 발동하는 버프들의 실전 가동률을 직접 입력하여 예측 정밀도 향상
4. **실시간 스탯 백분율(%) 안내**: 스탯 입력 패널 내에서 인게임 공식 기반 실제 효율/확률 수치를 실시간 표출
5. **셋팅 프리셋 비교**: 최대 3개까지 본인의 셋팅 조합을 임시 저장하고, 서로 간의 DPS 상승률(%)을 비교 분석
6. **모바일 최적화**: 모바일 브라우저에서도 한 손으로 편하게 터치하여 스탯과 룬을 조절할 수 있는 반응형 UI
## 🛠️ 수식 연산 및 스탯 효율 명세
인게임 공식 및 엑셀 v2.3 연산 수식을 기반으로 아래의 스탯 효율 환산 계산식을 적용하고 있습니다.
- **방어도 계수**: `1 / (1 + 방어수치 / 10328)`
- **스킬 개조 계수**: `1 + 0.03 * level + 0.02 * (보너스 단계 만족 여부)`
- **치명타 확률**: `0.5 * (치명타 스탯 / (치명타 스탯 + 2000)) + (허수아비류 타겟 시 0.3 보정)`
- **추가타 확률**: `추가타 스탯 / 13000` (1000당 7.69% 효율)
- **강타 / 연타 / 스킬위력 / 광역강화 효율**: `스탯 / 8500` (1000당 11.76% 효율)
- **콤보강화 효율**: `스탯 / 17500`
- **직접피해 DPS 기댓값**: `(1 + 주는피해 + 콤보피해) * (1 + 받는피해) * (1 + 강타피해 + 연타피해 + 추가타피해) * (치명타 보정값) * (추가타 보정값) * 공격력 * 2`
- **지속피해 DPS 기댓값**: `(1 + 주는피해) * (1 + 받는피해) * 지속피해계수 * 공격력 * 2`
## 🧠 인게임 데이터 정제 아키텍처 (Data Cleaning Pipeline)
룬 스크린샷 자동 OCR 텍스트 내 잔여 노이즈를 완벽하게 차단하기 위해 전문 정제 엔진을 구현하였습니다.
- **유니코드 특수기호 세척**: 한글, 영문, 숫자, 문장부호 이외의 기형적인 OCR 기호(`<`, `$`, `*` 등) 일괄 차단
- **맞춤법 및 오타 교정 사전**: `변화지` -> `변화를`, `피해지` -> `피해를` 등 OCR 어절 오타를 표준 한국어 표현으로 실시간 매핑 보정
- **블랙리스트 사전 (`NOISE_WORDS`)**: `전용 룬`, `판매 가능`, `용`, `다` 등 의미 없는 메타데이터 및 낱자 찌꺼기를 단어 길이별로 감지해 전역 소거
- **비탐욕적 정규식 매칭**: `1.5%` 의 소수점 때문에 문장이 조기 절단되어 찢기던 현상을 non-greedy 정규식으로 완벽 복원
